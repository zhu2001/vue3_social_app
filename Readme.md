vue3/nodjs/mongodb 搭建的社交网站

功能：登录注册，分享照片，心得，可关注其他用户，进入其他用户的主页。

在social-api 和 client 终端下执行npm i 安装依赖

在client中端下执行 npm run serve即可运行
